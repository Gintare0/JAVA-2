# GeoLocation

Programa, skirta geografinių koordinačių saugojimui ir atstumų tarp jų
skaičiavimui.

## Aprašymas

Sukurta klasė GeoLocation, kuri:
- saugo vietovės koordinates
- skaičiuoja atstumą tarp dviejų vietovių naudojant Haversino formulę
- skaičiuoja sukurtų objektų skaičių

Programa testuojama Main klasėje pagal pateiktą užduoties pavyzdį.

## Paleidimas

```bash
javac src/*.java
java Main
